# Fira Fonts

## Instruções para Download

### Fira Sans
Baixar de: https://fonts.google.com/specimen/Fira+Sans

Arquivos necessários:
- `FiraSans-Regular.ttf`
- `FiraSans-Medium.ttf`
- `FiraSans-SemiBold.ttf`
- `FiraSans-Bold.ttf`

### Fira Code
Baixar de: https://fonts.google.com/specimen/Fira+Code

Arquivos necessários:
- `FiraCode-Regular.ttf`
- `FiraCode-Medium.ttf`

## Nota

As fontes devem ser colocadas neste diretório (`assets/fonts/`) para serem carregadas pelo ThemeProvider via expo-font.

As fontes Fira Sans e Fira Code são gratuitas e open source sob SIL Open Font License.
